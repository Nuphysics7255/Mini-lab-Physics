🧠 Quiz Concept Check — ตัวเก็บประจุ (Capacitor)

ตอบแล้วมี Hint จนกว่าจะถูก • ระบบจะเก็บ error_map เพื่อทำ Dashboard

Σ สูตรสำคัญ — การชาร์จ/คายประจุของตัวเก็บประจุ

โทนส้ม = โหมดอธิบาย • เลื่อนอ่านได้ครบทั้งหน้า
แนวคิดหลัก

การชาร์จประจุ (Charging) คือการ “แยกประจุ” ให้แผ่นหนึ่งมีอิเล็กตรอน ขาด (จึงเป็น +) และอีกแผ่นมีอิเล็กตรอน เกิน (จึงเป็น −) โดย ประจุรวมของระบบไม่ถูกสร้างใหม่ (อนุรักษ์ประจุ)

การคายประจุ (Discharging) คือการที่อิเล็กตรอน “ไหลกลับ” เพื่อลดความต่างศักย์ระหว่างแผ่น จนสุดท้าย ความต่างศักย์เป็นศูนย์ แล้วการไหลหยุด

ในตัวนำโลหะ “อิเล็กตรอน” เป็นประจุที่เคลื่อนที่ได้ ส่วนโปรตอนอยู่กับที่ในโครงผลึกโลหะ

ประจุที่เก็บบนตัวเก็บประจุ

\[ Q = C V \]

ความหมาย: ประจุที่สะสมบนแผ่นมีขนาดสัมพันธ์กับความต่างศักย์ระหว่างแผ่น

ตัวแปร:
\(Q\) = ประจุที่เก็บ (คูลอมบ์, C)
\(C\) = ความจุไฟฟ้า (ฟารัด, F)
\(V\) = ความต่างศักย์ระหว่างแผ่น (โวลต์, V)

พลังงานที่สะสมในตัวเก็บประจุ

\[ U = \tfrac{1}{2} C V^2 = \tfrac{Q^2}{2C} = \tfrac{1}{2}QV \]

ความหมาย: พลังงานถูกเก็บใน “สนามไฟฟ้า” ระหว่างแผ่นตัวเก็บประจุ

ตัวแปร:
\(U\) = พลังงาน (จูล, J)

หยุดถ่ายเทเมื่อใด?

ตอนชาร์จ: กระแสจะค่อย ๆ ลดลง และหยุดเมื่อ

\[ V_{\text{capacitor}} = V_{\text{battery}} \]

ตอนคายประจุ: กระแสจะหยุดเมื่อความต่างศักย์ระหว่างแผ่นเป็นศูนย์

\[ V_{\text{capacitor}} \to 0 \]

หมายเหตุ: ใน Lab นี้เราเน้น “ภาพรวมแนวคิด” ของการแยกประจุและการกลับสู่สมดุล

สรุปเร็ว (จำให้ได้)

1) ในสายไฟเคลื่อนที่คือ อิเล็กตรอน
2) ชาร์จ = แยกประจุ (ไม่สร้างประจุใหม่)
3) หยุดชาร์จเมื่อ \(V_{cap}=V_{bat}\)
4) คายประจุหยุดเมื่อ \(V_{cap}=0\)
