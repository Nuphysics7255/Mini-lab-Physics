Mini Lab: ศักย์ไฟฟ้าในเปลือกทรงกลมตัวนำ
สำรวจประจุเหนี่ยวนำ ผิวใน–ผิวนอก ศักย์ไฟฟ้า และเงื่อนไข V = 0 เมื่อ grounded
กราฟ V-r และ E-r

สูตรสำคัญและขั้นตอนการคิด

สถานการณ์

มีประจุจุด q อยู่ที่ศูนย์กลางโพรงของเปลือกทรงกลมตัวนำหนา ผิวในรัศมี a ผิวนอกรัศมี b และกำหนดประจุสุทธิของเปลือกตัวนำเป็น Qshell

1) หลักการของตัวนำ

Econductor = 0
Vconductor = ค่าคงที่

ถ้าต่อสายดิน

Vshell = 0

2) การกระจายประจุบนผิว

ประจุที่ผิวในต้องทำให้สนามไฟฟ้าในเนื้อตัวนำเป็นศูนย์

Qinner = −q

ถ้าเปลือกมีประจุสุทธิ Qshell

Qouter = Qshell + q

3) ศักย์ไฟฟ้า กรณีไม่ต่อสายดิน

แบ่งตามตำแหน่ง r

เมื่อ r < a
V(r) = kqr − kqa + kQshell + qb
เมื่อ a ≤ r ≤ b
V(r) = kQshell + qb
เมื่อ r > b
V(r) = kQshell + qr

4) ศักย์ไฟฟ้า กรณีต่อสายดิน

โลกถ่ายเทประจุกับผิวนอกจนศักย์ของเปลือกเป็นศูนย์

Qouter = 0
เมื่อ r < a
V(r) = kqr − kqa
เมื่อ a ≤ r ≤ b
V(r) = 0
เมื่อ r > b
V(r) = 0

5) ขั้นตอนการคิด

  1. หา Qinner จากเงื่อนไข E = 0 ในเนื้อตัวนำ
  2. ใช้ประจุสุทธิของเปลือกเพื่อหา Qouter
  3. เลือกบริเวณของตำแหน่ง r
  4. รวมศักย์ไฟฟ้าแบบสเกลาร์
  5. ถ้าต่อสายดิน ใช้เงื่อนไข Vshell = 0
