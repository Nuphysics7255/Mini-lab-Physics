สัญญาณดิจิตอล 2 ระดับจากสัญญาณอะนาลอกที่มีช่วง -5V ถึง +5V

เส้นแดง = สัญญาณอะนาลอก (ต่อเนื่อง) • เส้นเขียว = สัญญาณดิจิตอล (0/1 หรือ -1/+1) • ใช้เกณฑ์ตัดสินจากแรงดันที่กำหนด

สถานะ: 0 (LOW)

ศึกษาเพิ่มเติมได้ที่เพจ คลายปมฟิสิกส์

อธิบายหลักการ: อะนาลอก vs ดิจิตอล และการตัดสินใจเป็น 0/1
1. อะนาลอกคืออะไร?
อะนาลอกคือสัญญาณที่เปลี่ยนค่าได้ต่อเนื่องตามเวลา เช่น เสียงพูด แรงดันในวงจร
2. ดิจิตอลคืออะไร?
ดิจิตอลคือสัญญาณที่มีจำนวนระดับจำกัด เช่น 0 และ 1 ทำให้ทน noise และประมวลผลได้ง่าย
3. กรณี 0 ถึง 5 V
ถ้าแรงดัน < เกณฑ์ → 0 (LOW) และถ้าแรงดัน > เกณฑ์ → 1 (HIGH)
4. กรณี -5 ถึง +5 V
ถ้าสัญญาณมีทั้งขั้วลบและบวก เราตั้งเกณฑ์ไว้ที่ 0V แล้วตีความได้ 2 แบบ
• แบบ 0/1 : มากกว่า 0 → 1, น้อยกว่า 0 → 0
• แบบ -1/+1 : มากกว่า 0 → +1, น้อยกว่า 0 → -1
แล้วแต่ระบบและเกณฑ์ที่กำหนด
5. กฎเกณฑ์การตัดสินใจ
กำหนด threshold ตัวเดียว เช่น \(V_{TH}\) แล้วเช็กว่าแรงดันสูงกว่าหรือต่ำกว่า ก็รู้เลยว่าเป็น 0 หรือ 1
6. ทำไมบางระบบใช้ +1 และ -1
เพื่อให้สัญญาณสมดุลรอบศูนย์ (เฉลี่ย ≈ 0) และคำนวณง่ายในงานสื่อสาร / DSP ถึงจะเป็น +1/-1 ก็ยังถือว่าเป็นดิจิตอล 2 ระดับอยู่ดี
แบบฝึกหัด: ดิจิตอล 2 ระดับ

เลือกคำตอบที่ถูกต้องที่สุดในแต่ละข้อ แล้วกด "ตรวจคะแนน"

ข้อที่ 1
ถ้าสัญญาณอยู่ในช่วง -5V ถึง +5V และตั้งเกณฑ์ไว้ที่ 0V แล้ววัดได้ +3V ระบบจะอ่านค่าเป็นอะไร?
ข้อที่ 2
ข้อใดอธิบายความแตกต่างของสัญญาณอะนาลอกและดิจิตอลได้ถูกต้องที่สุด
ข้อที่ 3
ระบบ 0–5V ตั้งเกณฑ์ 2.5V วัดได้ 1.0V ควรอ่านเป็นค่าใด?
ข้อที่ 4
สัญญาณที่เปลี่ยนระหว่าง 2 ระดับ คือ -1 กับ +1 ต่อเนื่องตลอดเวลา จัดเป็นสัญญาณแอนะล็อก
