สูตรสำคัญ: กล่องถูกดันติดผนัง แทนค่าปัจจุบัน

1. สมดุลแนวราบ (แกน x)

\(\sum F_x = 0 \Rightarrow N = F\)

แทนค่า: N = F = 20.00 N

2. น้ำหนักกับแรงเสียดทาน

\( mg = m g \)

แทนค่า: \(mg = m g\) = 9.80 N

\( f_s \le \mu_s N,\quad f_{s,\max} = \mu_s N\)

แทนค่า: \( f_{s,\max} = \mu_s N \) = 10.00 N

\( f_k = \mu_k N \)

แทนค่า: \( f_k = \mu_k N \) = 8.00 N

3. เช็คสถานะ: สมดุลหรือไถล?

ถ้า \( mg \le f_{s,\max} \) ⇒ กล่อง สมดุลแนวดิ่ง (ไม่ไถล) และ \( f = mg \)

ถ้า \( mg > f_{s,\max} \) ⇒ แรงเสียดทานสถิตไม่พอ กล่อง ไถลลง ใช้แรงเสียดทานจลน์ \( f = f_k = \mu_k N \)

ตอนนี้: สถานะ = Static, \( f = \)9.80 N

4. ถ้าไถล: หาความเร่งจากกฎของนิวตัน

แกนดิ่งลงเป็นบวก: \( \sum F_y = m a \)

\( mg - f = m a \Rightarrow a = g - \dfrac{f}{m} \)

ในกรณีใช้แรงเสียดทานจลน์: \( a = g - \dfrac{\mu_k N}{m} \)

แทนค่า: \( a = \)0.00 m/s²
