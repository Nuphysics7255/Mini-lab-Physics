  • ไฟฟ้ากระแสสลับมีทิศและขนาดเปลี่ยนตามเวลา

  • วงจรตัวต้านทานล้วนทำให้กระแสกับความต่างศักย์อยู่เฟสเดียวกัน
  • ในวงจรอนุกรม ต้องรวมความต้านทานก่อนหา Imax

สรุปสูตร (ไฟฟ้ากระแสสลับ)

  • ε(t) = εmax sin(ωt + φ)
  • I(t) = Imax sin(ωt + φ)
  • ค่า max คือ “ค่าสูงสุดของคลื่น” เช่น εmax, Imax
  • ค่า RMS คือค่าที่ให้กำลังไฟฟ้าเทียบเท่ากับไฟฟ้ากระแสตรง
  • Irms = Imax / √2
  • εrms = εmax / √2
  • Vrms = Vmax / √2
  • Pเฉลี่ย = Irms2R
  • หรือ Pเฉลี่ย = VrmsIrms
วิเคราะห์ข้อที่ควรทบทวน

ข้อควรระวัง
  • ค่า RMS ไม่ใช่ค่าเฉลี่ยเลขคณิตของคลื่นไซน์
  • ข้อนี้เป็นวงจรตัวต้านทานล้วน จึงไม่มีเฟสต่างกันระหว่างกระแสกับความต่างศักย์
  • หาค่าที่คร่อม R1 ต้องใช้ R1 = 3 Ω ไม่ใช่ Rรวม
  • กำลังเฉลี่ยต้องใช้ค่า RMS ไม่ใช่ค่าสูงสุด
