สูตรการคำนวณ

น้ำหนัก: W = mg = 5 × 10 = 50 N
แรงปฏิกิริยา: N = mg - F sin θ = 50 - 0 = 50 N
แรงเสียดทานสถิตสูงสุด: fs,max = μₛN = 0.50 × 50 = 25.0 N
แรงเสียดทานจลน์: fk = μₖN = 0.40 × 50 = 20.0 N
แรงลัพธ์สุทธิ: Fnet = Fx - f = 20.0 - 20.0 = 0.0 N
ความเร่ง: a = Fnet/m = 0.0/5 = 0.0 m/s²
แรงลัพธ์ (กรณีเคลื่อนที่แนวดิ่ง): Fnet = √(Fx2 + (Fy-mg)2) = 0.0 N
ความเร่งเชิงองค์ประกอบ: ax = Fx/m = 0.0 m/s², ay = (Fy-mg)/m = 0.0 m/s², a = 0.0 m/s²
