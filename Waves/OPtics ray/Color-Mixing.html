อธิบายหลักการผสมแสง (Additive) 💡

แนวคิดหลัก (สั้นๆ):
  • แสงทำงานแบบ “บวกความสว่าง” — เปิดไฟมากขึ้น = สว่างขึ้น
  • สีหลักของแสงมี 3 สี: R (แดง), G (เขียว), B (น้ำเงิน)
  • เมื่อแสงซ้อนกัน สีผสมจะเปลี่ยนตามสัดส่วนของ R, G, B
R + G = เหลือง G + B = ฟ้า (Cyan) B + R = ม่วงแดง (Magenta) R + G + B = ขาว
วิธีทดลองในหน้านี้:
  1. ใช้เมาส์/นิ้ว ลากวงแสง R, G, B ให้มาซ้อนกัน
  2. สลับสวิตช์ R/G/B เพื่อดูผลของการเปิด–ปิดแต่ละสี
  3. ปรับ รัศมี/ขอบนุ่ม/ความสว่าง แล้วสังเกตสีผสม
ตัวอย่างในชีวิตจริง: หน้าจอทีวี/มือถือ และโปรเจ็กเตอร์ผสมแสงแบบนี้เลย 🖥️✨

แบบฝึกหัด — ผสมแสง RGB 🎯

1) ถ้าเปิด R และ G แต่ปิด B สีในบริเวณซ้อนทับคืออะไร?

เฉลย: เหลือง เพราะ R + G = Yellow

2) ถ้าเปิด R, G, B พร้อมกันในบริเวณเดียวกัน ผลลัพธ์คือ?

เฉลย: ขาว เพราะการผสมแสงเป็นการ บวกความสว่าง ของ R, G, B

3) อยากได้ สีม่วงแดง (Magenta) ควรทำอย่างไร?

เฉลย: R + B จะได้ Magenta
