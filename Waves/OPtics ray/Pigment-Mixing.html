อธิบายหลักการผสมสารสี (Subtractive) 🎨

อธิบายหลักการ:
  • สารสี(pigments)เช่น สีทาบ้าน หมึกพิมพ์ หรือสีน้ำ สารสีจะทำหน้าที่ "ดูดซับ" หรือ "ลบ" แสงบางสีออกไป ทำให้เราเห็นแสงที่เหลืออยู่
  • พื้นขาวสะท้อนแสงทุกสี แต่ สารสีดูดกลืนแสงสี
  • แม่สีสารสีคือ CMY Cyan(สีฟ้าอมเขียว),Magenta (สีบานเย็น),และ Yellow (สีเหลือง) : C (ดูดแดง) , M (ดูดเขียว) , Y (ดูดน้ำเงิน)
  • ยิ่งทาทับหลายชั้น สีจะ เข้มขึ้น และ C+M+Y จะเข้าใกล้ ดำ
แสงสี vs สารสี (เข้าใจแบบง่าย ๆ):
  • แสงสี (Additive) ใช้แม่สี RGB = แดง (R) + เขียว (G) + น้ำเงิน (B)
  • ถ้ารวมแสง R + G + B ครบ จะเห็นเป็น แสงสีขาว (เพราะมีแสงทุกสีผสมกัน)
  • สารสี (Subtractive) ไม่ได้ “สร้างแสง” แต่ทำหน้าที่ ดูดกลืน/ลบ แสงบางสีออกไปจากแสงขาว
สารสีทำงานอย่างไร?
คิดภาพว่าเราเริ่มจาก แสงขาว ที่มี R+G+B ครบ แล้วสารสีจะ “ดูด” บางสีออก เหลือแสงที่สะท้อนกลับมาเข้าตาเรา เราจึงเห็นวัตถุเป็น “สีที่เหลือ” นั่นเอง
ตัวอย่างแสงสี: R+G=เหลือง, G+B=ฟ้า, R+B=ม่วงแดง และ R+G+B=ขาว
ตัวอย่างเชื่อมโยงกับสารสี: C ดูดแดง → เหลือ (G+B), M ดูดเขียว → เหลือ (R+B), Y ดูดน้ำเงิน → เหลือ (R+G) ดังนั้นทาซ้อนกันหลายชั้น แสงที่สะท้อนกลับมาน้อยลง → สี เข้มขึ้น และเข้าใกล้ ดำ
C + M = น้ำเงิน M + Y = แดง Y + C = เขียว C + M + Y ≈ ดำ
เชื่อมโยงชีวิตจริง:
  • เครื่องพิมพ์ ใช้หมึก CMYK เพื่อให้ดำสนิทและคุมการไล่ระดับได้ดี
  • สีน้ำ/สีโปสเตอร์ ทาทับหลายชั้นจะเข้มขึ้น เพราะแสงทะลุกลับได้น้อยลง
  • การทำโปสเตอร์ ลองทา C → ทับด้วย Y จะได้เขียวสดทันที

แบบฝึกหัด — ผสมสารสี CMY ✏️

1) Cyan + Yellow จะได้สีใดเด่น?

วิธีคิด: C ดูดแดง, Y ดูดน้ำเงิน เหลือการสะท้อนสีเขียว 🌿

2) Magenta + Cyan ควรได้สีใด?

วิธีคิด: M ดูดเขียว, C ดูดแดง → เหลือน้ำเงิน 💙

3) ถ้าทับ C + M + Y พร้อมกัน พื้นที่ซ้อนทับจะใกล้สีใด?

วิธีคิด: ดูดกลืนแสงทุกช่องสี จึงใกล้ดำ 🖤
