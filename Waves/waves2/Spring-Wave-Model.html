อนุภาคบนคลื่น — มวลติดสปริงหลายก้อน

แนวคิด: แต่ละก้อนมวลแทน อนุภาคของตัวกลาง ที่สั่นขึ้นลงรอบตำแหน่งสมดุล ส่วนรูปคลื่นที่เห็นคือ แบบแผนการรบกวน ที่เคลื่อนที่ไป ไม่ใช่มวลทั้งแถวไหลตามกัน
🧠 ดูการสั่นของอนุภาคแต่ละก้อน
สถานะ: พร้อมเริ่ม
สังเกต: แตะหรือคลิกก้อนใดก้อนหนึ่งเพื่อเลือก ก้อนที่เลือกจะเป็น สีแดง และกราฟด้านล่างจะแสดง y-t ของก้อนนั้น โดยใช้แกนบวกขึ้นด้านบนและลบลงด้านล่าง
📈 กราฟ y-t ของก้อนที่เลือก
ก้อนที่ 9 • สีแดง
กราฟนี้ช่วยย้ำให้เห็นว่าอนุภาคหนึ่งก้อนเพียงแค่สั่นขึ้นลงตามเวลา ไม่ได้วิ่งไปพร้อมรูปคลื่น

⚙️ แผงควบคุม

คาบ2.00 s
อัตราเร็วคลื่น180 px/s
สมการย่อy = A sin(kx - ωt)
ภาพที่ควรเห็นอนุภาคสั่นขึ้นลง

📘 สูตรสำคัญ

การสั่นของก้อนต้นกำเนิด yₛ(t) = A sin(ωt + φ)
สมการคลื่นแบบย่อ y(x,t) = A sin(kx - ωt + φ)
ความสัมพันธ์ของเลขคลื่นและความยาวคลื่น k = 2π / λ
ความถี่เชิงมุมและความถี่ ω = 2πf
อัตราเร็วคลื่น v = fλ
ถ้าเป็นคลื่นนิ่ง y(x,t) = 2A sin(kx) cos(ωt)
⚠️ จุดที่ต้องระวัง: เมื่อปรับเฟสเริ่มต้น ก้อน ต้นกำเนิด ควรยกหรือยุบทันทีตามค่า φ ส่วนก้อนอื่นยังอยู่ที่สมดุลจนกว่าคลื่นจะเดินทางไปถึง
⚠️ จุดที่ต้องระวัง: อนุภาคของตัวกลางไม่ได้เคลื่อนที่ไปกับคลื่น แต่สั่นรอบตำแหน่งสมดุล ขณะที่การรบกวนและพลังงานเป็นสิ่งที่ส่งต่อไปตามแนวคลื่น

📝 แบบฝึกคิด

1) ลองเพิ่มความถี่ แล้วสังเกตว่า คาบ เปลี่ยนอย่างไร
2) ลองเพิ่มความยาวคลื่น แล้วดูว่า ระยะระหว่างยอดคลื่น เปลี่ยนอย่างไร
3) เปลี่ยนเป็น คลื่นนิ่ง แล้วสังเกตตำแหน่งที่แทบไม่สั่น ว่าเป็น node
4) ถามนักเรียนว่า “ก้อนมวลทุกก้อนเคลื่อนที่ไปทางขวาพร้อมคลื่นหรือไม่” แล้วให้ตอบจากสิ่งที่เห็นจริง
