สูตรสำคัญ

สำหรับเกรตติงที่มีช่องจำนวนมาก ตำแหน่งแถบสว่างลำดับที่ n เป็นไปตามเงื่อนไข

d sinθ = nλ

หมายเหตุ: ในสูตรนี้ใช้ N เป็น “จำนวนเส้นต่อเมตร (เส้น/m)”
แต่ในสไลเดอร์จะใช้หน่วย “เส้น/mm” ซึ่งระบบจะแปลงให้โดยอัตโนมัติ

ถ้ารู้จำนวนเส้นต่อหนึ่งหน่วยความยาวของเกรตติง จะหาค่า d ได้จากสูตร

d = 1 / N

เมื่อมุมไม่ใหญ่มาก และฉากอยู่ห่างเกรตติงเป็นระยะ L จะประมาณตำแหน่งบนฉากได้ว่า

x ≈ L tanθ ≈ L nλ / d

ความหมายของตัวแปร

  • d = ระยะห่างระหว่างช่องของเกรตติง
  • N = จำนวนเส้นต่อหนึ่งหน่วยความยาวของเกรตติง
  • L = ระยะจากเกรตติงไปยังฉาก
  • λ = ความยาวคลื่นของแสง
  • n = ลำดับแถบสว่าง เช่น 0, ±1, ±2, ...

แนวคิดที่ควรสังเกต

  • ถ้า L มากขึ้น แถบสว่างบนฉากจะห่างกันมากขึ้น และฉากในรูปจะเลื่อนไปทางขวา
  • ถ้าจำนวนเส้นต่อมิลลิเมตรมากขึ้น จะทำให้ d เล็กลง และแถบสว่างห่างกันมากขึ้น
  • ถ้า d มากขึ้น แถบสว่างจะชิดกันมากขึ้น
  • ถ้า λ มากขึ้น แถบสว่างจะห่างออกจากแนวกึ่งกลางมากขึ้น
  • เมื่อเลือกแสงขาว แถบกลางยังเป็นสีขาว แต่แถบด้านข้างจะแยกสี และค่อย ๆ กระจายออกตามลำดับแถบ
