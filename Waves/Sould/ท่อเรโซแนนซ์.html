📘 สูตรสำคัญ — การสั่นพ้องของเสียงในท่อ (ปลายเปิด–ปลายปิด)

ปรากฏการณ์นี้เกิดจากอะไร?

เมื่อความถี่เสียงจากลำโพง “พอดี” กับความถี่ธรรมชาติของคอลัมน์อากาศในท่อ คลื่นเสียงที่สะท้อนกลับภายในท่อจะ เสริมกัน จนเกิด คลื่นนิ่ง ทำให้เสียงดังผิดปกติ เรียกว่า เรโซแนนซ์ (Resonance)

ภาพจำง่าย: ถ้าระยะทางไป-กลับ “เข้าจังหวะ” กัน คลื่นจะยิ่งพอกพูน → ดังขึ้น

เสียงจะดัง / เบา เมื่อไร?

ดังมาก เมื่อความยาวท่อทำให้เกิดคลื่นนิ่งชัดเจน (เรโซแนนซ์) พลังงานเสียงสะสมในท่อมาก → แอมพลิจูดสูง

เบา เมื่อความถี่ไม่ตรงเงื่อนไข คลื่นไป-กลับไม่เสริมกัน บางช่วงหักล้างกัน → แอมพลิจูดต่ำ

โมเลกุลอากาศข้างในเป็นอย่างไร?

เสียงเป็นคลื่นตามยาว: โมเลกุลอากาศสั่น ไป-กลับ ในแนวเดียวกับท่อ

คลื่นการกระจัด (Displacement) = มองการ “ขยับ” ของโมเลกุล

คลื่นความดัน (Pressure) = มองบริเวณ “อัดแน่น/เบาบาง” (ความดันสูง/ต่ำ)

สำคัญ: จุดที่การกระจัดมาก → ความดันเปลี่ยนแปลงน้อย และกลับกัน (ตำแหน่งสลับกัน)

สูตรเรโซแนนซ์

1) หลอดปลายปิด 1 ด้าน (ปลายเปิด–ปลายปิด)

\(L_{\text{eff}} = \dfrac{(2n-1)\,\lambda}{4}\),  \(n = 1,2,3,\dots\)L_eff = /(2n-1) λ4, n = 1,2,3,…

2) หลอดปลายเปิด 2 ด้าน (ปลายเปิด–ปลายเปิด)

\(L_{\text{eff}} = \dfrac{n\,\lambda}{2}\),  \(n = 1,2,3,\dots\)L_eff = /n λ2, n = 1,2,3,…

และความยาวคลื่น

\(\lambda = \dfrac{v}{f}\)λ = /vf

และอัตราเร็วเสียงในอากาศเมื่อทราบอุณหภูมิ

\(v = 331 + 0.6t\)v = 331 + 0.6t

โดย t คืออุณหภูมิในหน่วยองศาเซลเซียส (°C)

ระยะจาก “ดัง” ถึง “ดัง” ที่ติดกันห่างกันเท่าไร?

สำหรับทั้ง 2 กรณี (ปลายเปิด–ปิด และปลายเปิด–เปิด) ระยะเรโซแนนซ์ถัดไปห่างกัน

\(\Delta L = \dfrac{\lambda}{2}\)Δ L = /λ2

End correction คืออะไร?

ปากท่อที่ “เปิด” ไม่ได้จบการสั่นที่ขอบท่อจริง ๆ เพราะอากาศนอกปากท่อยังสั่นร่วมด้วยเล็กน้อย จึงเหมือนกับว่าท่อยาวกว่าความยาวจริง

\(L_{\text{eff}} = L + \Delta\)L_eff = L + Δ

สำหรับปลายเปิด 1 ด้าน โดยประมาณ

\(\Delta \approx 0.61\,r \approx 0.305\,D\)Δ ≈ 0.61 r ≈ 0.305 D

โดย \(r\) คือรัศมีท่อ และ \(D\) คือเส้นผ่านศูนย์กลางท่อ

สรุปสั้น ๆ

✅ ปรับ \(L\) หรือ \(f\) ให้เข้าเงื่อนไข → ดัง (เรโซแนนซ์)

✅ โหมด “การกระจัด” = ดูการขยับโมเลกุล • โหมด “ความดัน” = ดูอัดแน่น/เบาบาง

✅ ถ้าเปิด End correction จะใช้ \(L_{\text{eff}}\) แทน \(L\)

ข้อ 1
จากการทดลองเรื่องการกำทอนของเสียงโดยใช้หลอดกำทอน พบว่าเกิดกำทอนที่ระยะ 31.25 cm จากปากท่อ และเมื่อเลื่อนลูกสูบต่อไปอีกก็จะเกิดการกำทอนอีกครั้งที่ระยะ 43.75 cm ตามรูป
จงหาความถี่ของคลื่นเสียงที่ใช้
กำหนดให้ 1) อุณหภูมิอากาศ ณ ขณะนั้นเป็น 15 °C
2) ความเร็วเสียงในอากาศที่อุณหภูมิ 0 °C มีค่าเท่ากับ 331 m/s
ข้อ 2
หลอดกำทอนปลายเปิดทั้ง 2 ข้าง เมื่อเกิดกำทอนกับคลื่นเสียงที่มีความถี่ 350 เฮิรตซ์ ภายในหลอดจะมีตำแหน่งบัพกี่บัพ ถ้าหลอดยาว 1.5 เมตร และความเร็วเสียงในอากาศขณะนั้นเท่ากับ 350 เมตร/วินาที
ข้อ 3
ท่อทรงกระบอกปลายปิดข้างหนึ่งยาว 2 เมตร ความถี่ต่ำที่สุดของคลื่นเสียงที่ทำให้เกิดการสั่นพ้องในท่อนี้จะเท่ากับกี่เฮิรตซ์ กำหนดความเร็วของเสียงในอากาศเท่ากับ 340 เมตรต่อวินาที (ไม่ต้องคิด end-correction)
ข้อ 4
จากข้อ 3 ถ้าค่อย ๆ ปรับความถี่ให้สูงขึ้น จงหาความถี่ที่ทำให้เกิดเสียงดังครั้งที่สาม
