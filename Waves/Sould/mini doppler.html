ปรากฏการณ์ดอปเพลอร์ของเสียง
ความเร็วรถพยาบาล
0 m/s • รถหยุดนิ่ง
เลื่อนไปทางซ้าย = รถเคลื่อนไปทางซ้าย ←
เลื่อนไปทางขวา = รถเคลื่อนไปทางขวา →
ความสัมพันธ์ของความถี่ที่คนฟังได้ยิน
รถหยุดนิ่ง ⇒ fL = fs
โดยที่ fL = ความถี่เสียงที่คนฟังได้ยิน
fs = ความถี่ของแหล่งกำเนิดเสียง
สิ่งที่ควรสังเกต
• รถวิ่ง เข้าหาคน → หน้าคลื่นด้านหน้าจะถี่ขึ้น
→ fL > fs
• รถวิ่ง ออกห่างจากคน → หน้าคลื่นที่มาถึงคนจะห่างขึ้น → fL < fs
• ถ้ารถ หยุดนิ่ง → fL = fs
หมายเหตุ
กด เล่น ครั้งแรกเพื่อเปิดเสียงไซเรนของรถพยาบาล จากนั้นลองเลื่อนความเร็วไปทางซ้ายหรือขวา แล้วสังเกตทั้ง “ระยะหน้าคลื่น” และ “เสียงที่ได้ยิน”
