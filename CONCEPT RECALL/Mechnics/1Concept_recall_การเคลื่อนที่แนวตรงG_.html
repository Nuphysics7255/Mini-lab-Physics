สรุปผล

📘 สรุปสูตรและหลักการสำคัญในบทนี้

  • ระยะทาง = ความยาวเส้นทางที่เคลื่อนที่จริง
  • การกระจัด = การเปลี่ยนตำแหน่งแบบมีทิศทาง
  • อัตราเร็วเฉลี่ย = ระยะทางทั้งหมด / เวลาทั้งหมด
  • ความเร็วเฉลี่ย = การกระจัดทั้งหมด / เวลาทั้งหมด
  • ความเร่ง = \(\Delta v / \Delta t\)
  • กราฟ x–t → ความชัน = ความเร็ว
  • กราฟ v–t → ความชัน = ความเร่ง, พื้นที่ใต้กราฟ = การกระจัด
  • ถ้ามีพื้นที่ทั้งเหนือและใต้แกนเวลา → การกระจัด = พื้นที่เหนือแกน − พื้นที่ใต้แกน
  • กราฟ v²–s → ใช้สมการ \(v^2 = u^2 + 2as\) โดยความชัน = \(2a\)

🧠 ชุดสูตรการเคลื่อนที่แนวตรงที่ต้องจำ

  • \(v = u + at\)
  • \(s = \frac{u+v}{2}t\)
  • \(s = ut + \frac{1}{2}at^2\)
  • \(s = vt - \frac{1}{2}at^2\)
  • \(v^2 = u^2 + 2as\)

ข้อควรระวัง
  • อย่าสับสนระหว่าง ระยะทาง กับ การกระจัด
  • กราฟ v–t ต้องดูทั้ง ขนาด และ เครื่องหมาย ของพื้นที่
  • ที่จุดสูงสุดของการโยนขึ้นแนวดิ่ง v = 0 แต่ a = g ยังมีทิศลงล่างเสมอ
  • ถ้า v กับ a ทิศเดียวกัน → เร็วขึ้น, ถ้าสวนทางกัน → ช้าลง
