📘 สูตรสำคัญ: ขว้างจากตึก

โพรเจกไทล์คืออะไร

โพรเจกไทล์ คือการเคลื่อนที่ของวัตถุที่ถูก “ขว้าง/ยิง” แล้วลอยอยู่ในอากาศ โดย ไม่คิดแรงต้านอากาศ
ระหว่างลอยอยู่ในอากาศ แรงที่กระทำมีเพียง แรงโน้มถ่วง \((mg)\) ⇒ \(|\vec a| = g\) ตลอดเวลา
แนวระดับ: \(\sum F_x=0 \Rightarrow a_x=0 \Rightarrow v_x\text{ คงที่}\)
จุดสูงสุด: \(v_y=0\) แต่ \(|\vec v|_{\min}=v_x\) (ไม่เป็นศูนย์)
ใช้เวลา \(t\) เป็นตัวเชื่อมการคำนวณสองแกน

นิยามมุม (สำคัญ)

มุม \(\theta\) วัดจาก แนวราบไปทางขวา
มุมเงย: หมุนทวนเข็มนาฬิกา ⇒ \(\theta>0\)
มุมก้ม: หมุนตามเข็มนาฬิกา ⇒ \(\theta<0\)

แยกความเร็วต้น

\[ u_x = u\cos\theta,\qquad u_y = u\sin\theta \]
\[ \phi = \tan^{-1}\!\left(\frac{v_y}{v_x}\right) \]
มุม \(\phi\) คือมุมที่เวกเตอร์ความเร็ว \(\vec v\) ทำกับแกน \(x\) (วัดจากแกน \(x\) ไปยัง \(\vec v\))

สมการการกระจัด (Displacement)

\[ s_x = u_x t \]
\[ s_y = u_y t + \frac{1}{2} a_y t^2 \]
ถ้ากำหนดทิศขึ้นเป็นบวก จะได้ \(a_y=-g\)

ความเร็วเป็นฟังก์ชันของเวลา

\[ v_x = u_x,\qquad v_y = u_y + a_y t \]
\[ v = \sqrt{v_x^2+v_y^2} \]
หมายเหตุ: ในการทดลองนี้ไม่คิดแรงต้านอากาศ
📝 แบบฝึกหัด
เลือกคำตอบ A–D แล้วกด “ตรวจคำตอบ” เพื่อดูเฉลย (แสดงเฉพาะข้อที่เลือกแล้ว) • ตัวอักษรเฉลยเป็น สีแดง • กำหนดให้ใช้ \(g=10\ \text{m/s}^2\)

ข้อ 1) หากไม่คิดแรงต้านอากาศ ปริมาณใด “เป็นศูนย์” ตลอดการเคลื่อนที่?

เฉลย: A เพราะไม่คิดแรงต้านอากาศ ⇒ \(a_x=0\)

ข้อ 2) ที่จุดสูงสุดของโพรเจกไทล์ (ไม่คิดแรงต้านอากาศ) ข้อใดถูกต้อง?

เฉลย: C ที่จุดสูงสุด \(v_y=0\) แต่ \(v_x\) ยังไม่จำเป็นต้องเป็นศูนย์

ข้อ 3) ขว้างวัตถุจากตึก (ไม่คิดแรงต้านอากาศ) ข้อใดถูกต้องเกี่ยวกับ \(v_x\)?

เฉลย: B เพราะ \(a_x=0\) ⇒ \(v_x\) คงที่

ข้อ 4) ขว้างวัตถุออกไปจากหน้าผาสูง \(45\,\text{m}\) ในแนวระดับด้วยความเร็ว \(5\,\text{m/s}\) (กำหนด \(g=10\,\text{m/s}^2\)) วัตถุตกถึงพื้นล่างในเวลาเท่าใด?

เฉลย: B ใช้ \(t=\sqrt{\frac{2H}{g}}=\sqrt{\frac{2(45)}{10}}=\sqrt{9}=3\,\text{s}\)

ข้อ 5) หากไม่คิดแรงต้านอากาศ ข้อใดถูกต้องเสมอ?

เฉลย: C เพราะมีแรงเดียวคือ \(mg\) ⇒ \(\vec a\) ชี้ลงคงที่

ข้อ 6) ลูกบอลหลุดออกจากขอบโต๊ะที่สูง \(20\,\text{m}\) จากพื้น และตกกระทบพื้นที่จุดซึ่งห่างจากขอบโต๊ะในแนวระดับ \(2\,\text{m}\) (กำหนด \(g=10\,\text{m/s}^2\)) ความเร็วของลูกบอลขณะหลุดออกจากขอบโต๊ะมีค่าเท่าใด?

เฉลย: B เวลา \(t=\sqrt{\frac{2(20)}{10}}=\sqrt{4}=2\,\text{s}\) ⇒ \(u_x=\frac{2}{2}=1\,\text{m/s}\)

ข้อ 7) ขว้าง “ในแนวระดับ” จากตึก (มุม \(\theta=0\)) ข้อใดถูกต้อง?

เฉลย: B เพราะ \(u_y=u\sin0=0\)

ข้อ 8) ครูนุอยู่บนยอดตึกสูง \(50\,\text{m}\) แล้วขว้างลูกหินออกไปทำมุม \(37^\circ\) กับแนวระดับด้วยความเร็ว \(25\,\text{m/s}\) (กำหนด \(g=10\,\text{m/s}^2\)) นานเท่าไรลูกหินจึงจะตกถึงพื้นดิน?

เฉลย: B \(u_y=25\sin37^\circ\approx25(0.6)=15\). ใช้ \(0=50+15t-5t^2\) ⇒ \(t=5\,\text{s}\)

ข้อ 9) ความเร็วขณะกระทบพื้นหาได้จากข้อใด (เมื่อรู้ \(v_x, v_y\))?

เฉลย: B เพราะเป็นขนาดเวกเตอร์ความเร็ว

ข้อ 10) ขว้างลูกบอลด้วยความเร็ว \(10\,\text{m/s}\) จากยอดตึกสูง \(60\,\text{m}\) โดยทำมุมเงย \(30^\circ\) กับแนวระดับ (กำหนด \(g=10\,\text{m/s}^2\)) จงหาว่าลูกบอลตกห่างจากตึกประมาณเท่าใด?

เฉลย: C \(u_x=10\cos30^\circ\approx8.66\,\text{m/s}\), \(u_y=10\sin30^\circ=5\,\text{m/s}\). แกนดิ่งใช้ \(0=60+5t-5t^2\) ⇒ \(t=4\,\text{s}\). ดังนั้น \(S_x=u_x t\approx8.66(4)=34.64\approx35\,\text{m}\)
