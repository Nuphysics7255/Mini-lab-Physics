มุมปัจจุบัน: 60.0°
สถานะ: สมดุล
มุมคาน = 60.0°
มุมที่น้อยที่สุดที่ยังไม่ไถล = คำนวณ...
Ff ที่พื้น = - N
ขีดจำกัดที่พื้น = - N
N (ผนัง) = - N
แรงเสียดทานที่ผนัง = - N (ทิศขึ้น)
ตำแหน่ง cm = กลางคาน
ศึกษาเพิ่มเติมได้ที่ เพจคลายปมฟิสิกส์

วิธีคำนวณสมดุล และหามุมที่น้อยที่สุดที่ยังไม่ไถล

1) สมดุลในแนวนอน

$$\sum F_x = 0 \;\Rightarrow\; F_f = N_w$$

2) สมดุลในแนวดิ่ง

$$\sum F_y = 0 \;\Rightarrow\; N_f + F_w = W$$

และ $$F_w = \mu_{\text{wall}}\, N_w$$

3) สมดุลโมเมนต์ที่ปลายล่างของคาน

$$N_w \, h + F_w \, b = W \,(b f)$$

โดย \(b = L \cos\theta,\; h = L \sin\theta\)

เมื่อคาน “จะเริ่มไถล” ที่พื้น

$$F_f = \mu_{\text{floor}}\, N_f$$

และจากข้อ 1) จะได้ว่า

$$N_w = \mu_{\text{floor}} (W - F_w)$$

จับคู่กับนิพจน์ \(N_w\) จากโมเมนต์

$$N_w = \frac{b f W}{h + b \mu_{\text{wall}}}$$

ตัด \(W\) และแทน \(b = L\cos\theta, h = L\sin\theta\) จะได้

$$\frac{f\cos\theta}{\sin\theta + \mu_{\text{wall}}\cos\theta} = \frac{\mu_{\text{floor}}}{1 + \mu_{\text{floor}}\mu_{\text{wall}}}$$

จัดรูป

$$\tan\theta_{\min} = \frac{f(1 + \mu_{\text{floor}}\mu_{\text{wall}}) - \mu_{\text{floor}}\mu_{\text{wall}}}{\mu_{\text{floor}}}$$

$$\theta_{\min} = \tan^{-1}\left( \frac{f(1 + \mu_{\text{floor}}\mu_{\text{wall}}) - \mu_{\text{floor}}\mu_{\text{wall}}}{\mu_{\text{floor}}} \right)$$

*ในที่นี้ “น้อยที่สุดที่ยังไม่ไถล” คือมุมวิกฤตที่ถ้ายกชันกว่านี้ (มุมมากขึ้น) พื้นจะเริ่มต้านไม่ไหว
